# Biomedical Data Science, Mining and Modeling

Course projects exploring biomedical data mining, machine learning, network analysis, and computational modeling techniques. This is part of the CB&B 7520 course at Yale University. 

## Homework 1: Protein Interaction Networks

**Folder:** `homework-1-protein-interaction-networks`

Topics explored:

- Protein-protein interaction (PPI) networks
- Graph-based biological analysis
- Network connectivity and topology
- Biological pathway interpretation
- Data visualization in Python
- Biomedical network analysis

## Homework 2: Single Cell Analysis

**Folder:** `homework-2-single-cell-analysis`

Topics explored:

- Single-cell RNA sequencing analysis
- Dimensionality reduction
- Clustering and visualization
- Gene expression analysis
- Cell population identification
- Computational biology workflows

## Homework 3: Network and Pathway Analysis

**Folder:** `homework-3-network-and-pathway-analysis`

Topics explored:

- Biological network analysis
- Pathway enrichment methods
- Gene interaction modeling
- Graph algorithms in biomedicine
- Systems biology concepts
- Biomedical data science workflows
  
## Tools & Libraries

Some projects use:

- Python
- NumPy
- pandas
- matplotlib
- scikit-learn
- NetworkX
- Scientific Python workflows

These projects were completed as part of coursework in biomedical data science, computational biology, and machine learning.
